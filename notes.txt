1. npm init -y
2. change in package.json "type": "module",
3. install typescript , tsx as dev dep. -> npm i -D typescript tsx
4. npx tsc --int -> to create tsconfig.json
5. adjust rootDir="./src" and outDir="./dist" in tsconfig.json
6. start the dev server with npm run dev
7. to make build use npm run build
8. to start the build using npm run start
